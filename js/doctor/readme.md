js是一门高级语言，被计算机cpu执行前，需要通过某种程序将js转化为低级的机器语言并执行，这种程序就是js引擎

v8引擎是js引擎的一种，是一个接收js代码，编译代码然后执行的c++程序，编译后的代码可以在多种操作系统，多种处理器上运行
